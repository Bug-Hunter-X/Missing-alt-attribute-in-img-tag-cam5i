This is a common mistake when using the <img> tag in HTML. The alt attribute is crucial for accessibility and SEO. It provides a text alternative for the image, which is important for users with visual impairments who use screen readers. Additionally, search engines use the alt text to understand the image's content.

The error occurs when the alt attribute is either missing or left empty.

<img src="image.jpg">

This code is incorrect as the image lacks alt text.
